Detailed Description

Son program will count the number of 'F' characters which represent fire in the forest. The Son program will take one input parameter by command line, the input is a string of charecters 'G','F','T' which represnts a forest.

Son program is resposible to count number of Fires which are represented by 'F' character in a string. The program is intended to take the input parameters from command line. The only allowed number of arguments is two: 1 for the program name (Son.exe) and 1 for the Forest String Which will be processed by the program and count the number of Fires.

The program will return the number of Fires using exit value. If the program exits with (-1), the count didn't occure correctly.
