Introduction to Operating Systems - EX1.

Ex1 has two different projects, each defines a seperate program. The "Father" project, and the "Son" project. The main idea is to practice the concept of process calling using the supplied API by the OS.

The "Father" program will call the "Son" program as a child process, and will use its return value to complete its task.

Full documentation of implemntation is found as HTML under (Docs/html/index). If reading in github: Online version: https://levymatan.github.io/ISP_ex1_doc/
